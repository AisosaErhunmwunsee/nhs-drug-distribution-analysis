# NHS Drug Distribution Equity Analysis

##  Overview

This project explores healthcare inequalities and medicine distribution patterns using simulated NHS prescribing data inspired by real-world NHS healthcare analytics challenges.

The analysis investigates:

- Medicine shortages
- Regional inequalities
- Pharmacy access pressures
- Deprivation-linked healthcare disparities
- Forecasting approaches for healthcare planning

---

##  Analytical Focus

The project demonstrates:

- Data cleaning

- Healthcare data visualisation
- Correlation analysis
- ARIMA forecasting
- Basic predictive modelling

---

## Key Findings

- Deprivation showed a positive relationship with medicine shortages.
- Rural regions experienced greater shortage pressures.
- Healthcare access patterns varied across localities.
- Predictive analytics demonstrated potential for proactive planning.
- ARIMA forecasting illustrated future shortage trend estimation.
